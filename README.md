# Algorithmic Trading Project

## Overview
This project implements an algorithmic trading system that fetches historical candle data for various financial instruments using the Zerodha Kite API. The data can be used for backtesting trading strategies and analyzing market trends.

## Features
- Fetch historical candle data for different financial instruments.
- Support for date range selection to get specific data periods.
- Data is returned in a structured format for easy analysis and visualization.

## Prerequisites
Before you begin, ensure you have the following installed:
- Python 3.6 or later
- `requests` library
- `pandas` library
